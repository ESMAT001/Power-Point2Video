# Power-Point2Video
Change power-point files  to video slideshow with voice

NOTE : This app requires [LibreOffice](https://www.libreoffice.org/download/download/) to be installed in the system .
<br/>
NOTE : Set your [Play.ht](https://play.ht/) credentials in frontend/src/utils.js file.

Start Frontend Server :
```
cd frontend
yarn start
```

Start Backend Server :
```
cd backend
yarn start  
```
